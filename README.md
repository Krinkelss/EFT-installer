# EFT-installer
Небольшая программка для устранения одной из причин невозможности запуска Таркова на SPT. 
Версия через bat файл

# Что делать и кто будет в этом виноват?
Кидаем в папку с игрой, и запускаем наш bat файл. Он автоматически создаст необходимые файлы и сделает запись в реестр. Наслаждаемся игрой.
Если опять не получилось запустить, ну, значит не судьба
